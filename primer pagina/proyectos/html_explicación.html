inicio

html

empecemos por lo primero. HTML, ¿que significa?
bueno, html es la abreviacion de HyperText markup language, lo que se traduce al español como lenguaje de marcacion de hipertexto.
basicamente consta de un sistema de etiquetas que le dicen al interprete (el navegador) que es cada cosa, con estas etiquetas podemos especificar si un texto es un titulo, si es que tiene que ir en negrita , en mayúsculas etc.

CSS

ok, html nos sirve para darle estructura a una pagina web pero.. solo eso no son mas que palabras con una sierta indicacion, para darle vida, color y basicamente estilo a la pagina se usa:
CSS que significa cascading style sheets, en español, hojas de estilo en cascada.
aunque a travez de html se puede aplicar estilos usamos css para apartar la estructura de la apariencia y ademas en html no tenemos todo lo que en css

JavaScript

finalmente tenemos JavaScript, este lenguaje es lo que le va a dar a la pagina una verdadera interfaz con cosas que se muevan y siertas acciones programables. es una herramiena importante a la hora de hacer que la pagina sea dinamica
